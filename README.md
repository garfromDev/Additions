# Additions
Vérification d'additions en écriture libre

## Généralité
Application web destinée à être utilisée sur un téléphone dans le navigateur.

## Fonctionnement
A partir de l'application, l'utilisateur utilise la caméra pour prendre en photo une addition. écrite manuellement. l'application décode l'addition, effecture le calcul etindique : 
-  le résultat de l'addition
- si le résultat est identique ou non au résultat obtenu par celui qui a posé l'addition manuellement

voir ![addition_juste](./addition_juste.png) et [addition_fausse](./addition_fausse.png)


## contrainte impérative
Doit fonctionner avec le navigateur safari sous iOS

## objectifs (si possible)
l'application fonctionne sans appels à des API ou sites extérieurs.
l'application tourne entièrement dans le navigateur (pas de backend)
l'application est un fichier html contenant le code et les styles, il suffit de télécharger le fichier sur le téléphone pour utiliser l'appli.
